# MIKATYPE

美佳のタイプトレーナー JAVA版

## Usage

```sh
javac MIKATYPE.java
java MIKATYPE
```

## Setup

install openjdk@17
```sh
brew install openjdk@17
echo 'export PATH="/opt/homebrew/opt/openjdk@17/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc
```

## Data

- IchigoJam練習モード from [IchigoJam BASIC リファレンス](https://github.com/IchigoJam/doc)
